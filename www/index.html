Plugins for Q3Radiant 200 - Thursday, June 25, 2000
Here is the current list of plugins publicly available for Q3Radiant 200. If you are writing a plugin for Q3R and are not listed here let me know so I can add you. Plugin SDK for 200 will be released shortly.

bobToolz - detect overlapping brushes
Curry - shader preview and editing
Q3Build - a frontend to the BSP tools
GenSurf - surface generator
TexTool - Texture manipulation tools (online documentation)
PrtView - Portal viewer for map optimisations
Pk3Man - A pk3 manipulation plugin
MGS - all-purpose tools for MAP manipulations (no official release yet)

See the plugins file section for some downloads.
